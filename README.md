# bridge-medisante

Bridge for Médisanté devices.

Deployed on bridge-medisante.pryv.net

Prerequisites: Node v8+, Yarn v1+

## How to?

| Task                              | Command                        |
| --------------------------------- | ------------------------------ |
| Setup                             | `yarn install`                 |
| Create Distribution               | `yarn release`                 |
| Run API server                    | `yarn start`                   |
| Run Tests                         | `yarn test`                    |
